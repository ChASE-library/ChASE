Installation and Setup on a Cluster¶

This section provides guidelines to install and setup the ChASE library on computing clusters. Each guideline is given as a step by step set of instructions to install ChASE on a cluster either with or w/o support for GPUs. After the setup of ChASE, users are provided with a description of how to link and integrate ChASE into their own codes. Such integration can be achieved either by using one of the interfaces provided by the library, or by following our instructions to implement the user’s own interface.
