
ChaseMpiDLACudaSeq
^^^^^^^^^^^^^^^^^^^^^^^^^

``ChaseMpiDLACudaSeq`` defines the implementation of ``ChaseMpiDLAInterface``
for single node system without MPI using 1 GPU card.  


.. note::
    For more details of the implementation of ChASE-MPI, please refer to :ref:`chase-single-gpu`
    in the Developer Documentation.
